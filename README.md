# eco-track

EcoTrack is a gamified campus sustainability platform designed to convert sustainable actions into a fun, rewarding habit.

## Features
- **Dashboard**: Track your total points, CO2 avoided, and current streak.
- **Activities**: Log eco-friendly activities to earn points and save CO2.
- **Leaderboard**: Compete with other campus users.
- **Rewards**: Claim badges based on points and activities logged.
- **Profile**: View equivalent stats (trees planted, km not driven) and your badge collection.

## Getting Started
1. `npm install`
2. `npm run dev`
